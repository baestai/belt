3선탄 통합관리

현장에서 누구나 쉽게 쓰는 월간 점검 관리 도구입니다. 캘린더로 점검 일정을 확인하고, 휴대폰으로 바로 점검 결과를 입력하세요.

앱 열기 → 사용법 보기
두 가지 모드 점검모드 점검 입력 관리모드 점검 포인트 백업 · 동기화 자주 묻는 질문

🚀 빠르게 시작하기

설치가 필요 없습니다. 휴대폰·PC 브라우저에서 주소만 열면 바로 사용할 수 있습니다.

  1. 앱을 엽니다. 처음 화면은 점검모드(캘린더)입니다.
  2. 날짜를 선택합니다. 그 날 점검할 벨트 목록이 위쪽에 표시됩니다.
  3. 벨트를 눌러 점검을 입력합니다. 항목별로 정상/불량을 체크하고 저장하면 끝입니다.

🔀 두 가지 모드

현장 작업용과 관리자용으로 화면이 나뉩니다.

점검모드

비밀번호 없이 누구나 사용. 캘린더에서 점검 대상을 확인하고 결과를 입력하는 현장 화면입니다.

관리모드

관리자 비밀번호 입력 후 진입. 벨트·점검자·일정 편성·보고서·백업 등 설정을 관리합니다.

📅 점검모드 (현장)

현장에서 가장 많이 쓰는 화면입니다.

🗓 점검 캘린더

달력의 점으로 그날 점검 상태를 한눈에 봅니다. 날짜를 누르면 해당일 점검 대상이 위에 나타납니다.

👤 점검자 고정 (이 기기)

휴대폰에 본인 이름을 저장하면 다시 바꾸기 전까지 점검자가 자동 입력됩니다. 여러 명이 함께 쓰는 PC에서는 고정하지 않는 것을 권장합니다.

💬 빠른 메모 · 🎤 음성 입력

자주 쓰는 메모는 칩을 눌러 바로 입력. 마이크 버튼으로 음성을 받아 적을 수도 있습니다.

🕒 전월 대비 비교

지난 점검 결과를 함께 보여줘 변화가 있었는지 바로 비교할 수 있습니다.

📋 점검 입력하기

벨트를 선택하면 항목별 점검 폼이 열립니다.

  1. 점검자와 날짜를 확인합니다. 기기에 고정된 점검자가 있으면 자동으로 채워집니다.
  2. 항목별로 상태를 체크합니다. 정상/불량을 누르고, 필요하면 메모·음성으로 특이사항을 남깁니다.
  3. Pulley · 전기장치를 추가/삭제할 수 있습니다. 벨트마다 점검 항목 구성을 현장에서 직접 조정합니다.
  4. 점검 완료 저장을 누릅니다. 저장 후 캘린더로 돌아오며 상태 점이 갱신됩니다.
💡 점검표 PDF 저장: 벨트 상세 화면에서 “점검표 인쇄 / PDF 저장”을 누르면 인쇄 대화상자가 열립니다. 대상을 PDF로 저장으로 선택하면 파일로 보관할 수 있습니다.

🔐 관리모드

관리자 비밀번호를 입력하면 진입합니다. 설정과 데이터를 관리하는 공간입니다.

🧱 벨트 관리

벨트 추가·수정·삭제 및 구역(그룹) 분류 관리.

👷 점검자 관리

점검자 명단을 추가·삭제합니다.

💬 빠른 메모 관리

현장에서 쓰는 빠른 메모 칩을 직접 추가·삭제합니다.

🗓 점검일 편성

벨트별 월간 점검 일정을 캘린더에 배정합니다.

📄 보고서 (엑셀)

월간 점검 결과를 표로 정리해 내보냅니다.

🏆 점검 포인트 랭킹

점검자별 누적 포인트 TOP 10을 확인합니다.

💾 데이터 백업 / 복원

전체 데이터를 JSON 파일로 내보내고 다시 불러옵니다.

📑 구성 복사

한 벨트의 점검 항목 구성을 다른 벨트로 복사합니다.

🏆 점검 포인트 & 랭킹

점검 활동을 게임처럼 누적 관리해 적극적인 점검을 응원합니다.

① 기본 점수

점검 1건을 완료할 때마다 기본 포인트가 쌓입니다.

② 이상 발견 보너스

이상(불량)을 발견해 기록하면 추가 포인트를 받습니다. 꼼꼼한 점검을 보상합니다.

전체 누적과 이번 달 기준을 탭으로 나눠 보며, 🥇🥈🥉 메달과 함께 TOP 10을 표시합니다.

💾 데이터 저장 · 동기화

데이터는 자동으로 저장됩니다. 기기 간 공유는 클라우드 연결 시 가능합니다.

📦 자동 저장

입력한 내용은 브라우저에 자동 보관되어 새로고침해도 유지됩니다.

☁️ 클라우드 동기화

클라우드가 연결되면 여러 기기에서 같은 데이터를 함께 사용할 수 있습니다.

💾 수동 백업

관리모드에서 JSON 파일로 내보내 안전하게 보관하고, 필요할 때 복원합니다.

📱 기기별 설정

‘점검자 고정’은 기기마다 따로 저장되어 다른 기기에 영향을 주지 않습니다.

❓ 자주 묻는 질문

비밀번호를 잊었어요

관리모드 진입과 일부 변경에는 관리자 비밀번호가 필요합니다. 관리자에게 문의하세요.

음성 입력이 안 돼요

브라우저가 음성 인식을 지원해야 합니다. 최신 Chrome 사용을 권장하며, 마이크 권한을 허용하세요.

다른 기기에서 데이터가 안 보여요

클라우드가 연결되지 않은 경우 데이터는 해당 기기에만 저장됩니다. 백업/복원으로 옮길 수 있습니다.

실수로 저장했어요

벨트 상세의 점검 이력에서 해당 기록을 다시 열어 수정할 수 있습니다.
