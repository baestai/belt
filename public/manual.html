🚀 빠르게 시작하기

설치가 필요 없습니다. 휴대폰·PC 브라우저에서 주소만 열면 바로 사용할 수 있습니다. 홈 화면에 추가하면 앱처럼 쓸 수 있어요(PWA).

  1. 앱을 엽니다. 처음 화면은 홈(현황판)입니다. 금일 근무·점검 현황과 이상 설비를 한눈에 봅니다.
  2. 점검모드로 이동합니다. 하단 탭에서 점검모드를 누르고, 벨트/집진기 탭과 날짜를 고르면 그날 점검 대상이 표시됩니다.
  3. 설비를 눌러 점검을 입력합니다. 항목별로 정상/불량을 체크하고 저장하면 끝입니다.
🕖 점검일은 07시 기준으로 바뀝니다. 주간 근무 시작 시각에 맞춰, 새벽 07시 이전까지는 ‘전날’ 점검일이 유지됩니다. (예: 23일 06:59 → 22일 점검, 23일 07:00 → 23일 점검)

🧭 3개 탭으로 구성

하단의 세 탭으로 화면이 나뉩니다.

🏠 홈

금일 근무·점검 현황, 누적 점검 통계, 이상 설비·수리 상태를 모아 보는 현황판입니다.

🦺 점검모드

비밀번호 없이 누구나 사용. 벨트·집진기 캘린더에서 대상을 확인하고 결과를 입력하는 현장 화면입니다.

📊 관리모드

관리자 비밀번호 입력 후 진입. 설비·점검자·일정·보고서·정비 이력·백업 등을 관리합니다.

🏠 홈 대시보드

앱을 열면 가장 먼저 보이는 현황판입니다.

🔁 금일 근무현황

오늘 A~D조의 주간/야간/휴무와 대근 교체 인원을 한 줄로 보여줍니다.

📋 금일 점검현황

오늘 예정 대비 완료한 벨트·집진기 점검 수와 정상/이상 건수를 카드로 표시합니다.

🍩 누적 점검 통계

지금까지 점검한 벨트·집진기의 정상/이상 비율을 도넛 그래프로 보여줍니다.

⚠️ 누적 이상 목록

이상이 남은 설비를 모아 보여줍니다. 항목을 펼쳐 정비의뢰/수리완료를 관리하고, 📋 복사로 수리요청 문구를 한 번에 복사합니다.

🏆 홈 우상단의 Top10(점검 포인트 랭킹)·대근 아이콘으로 바로 이동할 수 있습니다.

📅 점검모드 (현장)

현장에서 가장 많이 쓰는 화면입니다. 상단 탭으로 🦺 벨트 / 🌀 집진기를 전환합니다.

🗓 점검 캘린더

달력의 점으로 그날 점검 상태를 한눈에 봅니다. 날짜를 누르면 해당일 점검 대상이 위에 나타납니다.

🔍 검색 · 상태 필터

설비명을 검색하거나 정상/이상/미점검 통계를 눌러 해당 설비만 골라 봅니다.

👤 점검자 고정 (이 기기)

휴대폰에 본인 이름을 저장하면 다시 바꾸기 전까지 점검자가 자동 입력됩니다. 공용 PC에서는 고정하지 않는 것을 권장합니다.

💬 빠른 메모 · 🎤 음성 입력

자주 쓰는 메모는 칩을 눌러 바로 입력. 마이크 버튼으로 음성을 받아 적을 수도 있습니다.

📋 점검 입력하기

설비를 선택하면 항목별 점검 폼이 열립니다. 벨트와 집진기는 점검 항목이 다릅니다.

🦺 벨트 점검

벨트 상태·마모, Pulley(베어링 상태 + 좌우 온도), 모터, 감속기, 전기장치, 급유·급지 등을 항목별로 점검합니다.

🌀 집진기 점검

차압, 탈진(펄스), Main Fan(Impeller/Damper, Ampere·진동), 주요 장치(Chamber·Duct·Hood·후처리·외관)를 점검합니다.

  1. 점검자와 날짜를 확인합니다. 기기에 고정된 점검자가 있으면 자동으로 채워집니다.
  2. 항목별로 상태를 체크합니다. 정상/불량을 누르고, 온도·진동·차압 등 수치를 입력하고, 필요하면 메모·음성으로 특이사항을 남깁니다.
  3. 구성 항목을 추가/삭제할 수 있습니다. 벨트마다 Pulley·전기장치 구성을 현장에서 직접 조정합니다.
  4. 점검 완료 저장을 누릅니다. 저장 후 캘린더로 돌아오며 상태 점이 갱신됩니다.
💡 점검표 PDF 저장: 벨트 상세·집진기 점검 화면에서 “점검표 인쇄 / PDF 저장”을 누르면 인쇄 대화상자가 열립니다. 대상을 PDF로 저장으로 선택하면 파일로 보관할 수 있습니다.

🔧 수리요청 · 정비 이력

발견한 이상을 정비로 연결하고, 처리 과정을 추적합니다.

  1. 홈 ‘누적 이상 목록’에서 이상 항목을 펼칩니다. 어떤 부위가 불량인지, 점검 메모와 함께 보입니다.
  2. 📋 복사로 수리요청 문구를 보냅니다. 설비·이상 항목·점검일이 정리된 문구가 클립보드에 복사됩니다.
  3. 정비의뢰로 진행 상태를 표시합니다. 담당자·예상일을 함께 기록할 수 있습니다.
  4. 수리가 끝나면 수리완료를 누릅니다. 점검 상태가 양호로 바뀌고, 내용이 정비 이력 대장에 보관됩니다.
📒 관리모드의 🔧 정비 이력에서 완료된 수리를 설비별로 조회할 수 있습니다(담당자·점검일·완료일 포함).

📈 측정값 추이

반복 점검에서 쌓인 수치의 변화를 선 그래프로 보여줍니다(점검 2회분 이상부터).

🌡 Pulley 최고온도

벨트 상세 화면에서 점검할 때마다의 Pulley 최고온도(℃) 변화를 추적해 과열 경향을 미리 파악합니다.

💨 집진기 차압

집진기 점검 폼에서 차압(㎜Aq) 추이를 보여줘 여과포 막힘 등 변화를 빠르게 확인합니다.

🔐 관리모드

관리자 비밀번호를 입력하면 진입합니다. 종합 상태는 정상/이상/미점검으로 표시됩니다.

📂 설비가 많아, 카테고리(구역·집진기)를 누르면 해당 설비 목록이 표시됩니다. 검색창에 설비명을 입력하면 벨트·집진기를 한 번에 찾을 수 있습니다.

🧱 벨트 추가

구역(그룹)을 지정해 벨트를 추가합니다. 삭제·구성 복사는 벨트 상세에서.

🌀 집진기 관리

집진기와 월간 점검일을 추가·수정·삭제합니다.

👷 점검자 관리

점검자 명단을 추가·삭제합니다.

🔁 교대조 편성

A~D조 인원과 PIN 초기화 승인을 관리합니다.

💬 빠른 메모

현장에서 쓰는 빠른 메모 칩을 직접 추가·삭제합니다.

📄 보고서 (엑셀)

월 범위·점검자·이상만 등으로 걸러 점검 결과를 CSV로 내보냅니다.

🔧 정비 이력

완료된 수리(정비)를 설비별로 조회합니다.

📋 변경 이력

점검 저장·벨트 추가/삭제·정비의뢰/수리완료를 누가·언제 했는지 기록합니다.

🏆 점검 포인트 랭킹

점검자별 누적 포인트 TOP 10을 확인합니다.

💾 백업 / 복원

전체 데이터를 JSON으로 내보내거나, 자동 스냅샷 시점으로 복원합니다.

🏆 점검 포인트 & 랭킹

점검 활동을 게임처럼 누적 관리해 적극적인 점검을 응원합니다.

① 기본 점수

점검 1건마다 기본 10점. 단 CWF 벨트는 1건당 1점으로 계산합니다.

② 이상 발견 보너스

이상(불량)을 발견해 기록하면 1건당 2점을 추가로 받습니다. 꼼꼼한 점검을 보상합니다.

전체 누적과 이번 달 기준을 탭으로 나눠 보며, 🥇🥈🥉 메달과 함께 TOP 10을 표시합니다.

💾 데이터 저장 · 동기화

데이터는 자동으로 저장되고, 클라우드 연결 시 여러 기기가 실시간으로 함께 씁니다.

☁️ 클라우드 동기화

여러 기기·사람이 같은 데이터를 실시간으로 공유합니다. 한 곳의 입력이 곧바로 반영됩니다.

📦 자동 저장

입력한 내용은 브라우저에도 자동 보관되어 새로고침해도 유지됩니다(오프라인 대비).

🕒 자동 백업 스냅샷

12시간마다 이 기기에 상태 스냅샷이 저장됩니다. 백업 화면에서 원하는 시점으로 되돌릴 수 있습니다.

💾 수동 백업

관리모드에서 JSON 파일로 내보내 안전하게 보관하고, 필요할 때 복원합니다.

❓ 자주 묻는 질문

오늘 날짜가 어제로 보여요

점검일은 07시 기준으로 갱신됩니다. 새벽 07시 이전에는 전날 점검일이 유지되는 정상 동작입니다.

비밀번호를 잊었어요

관리모드 진입과 일부 변경에는 관리자 비밀번호가 필요합니다. 관리자에게 문의하세요.

음성 입력이 안 돼요

브라우저가 음성 인식을 지원해야 합니다. 최신 Chrome 사용을 권장하며, 마이크 권한을 허용하세요.

실수로 저장했어요

벨트 상세·집진기의 점검 이력에서 해당 기록을 다시 열어 수정할 수 있습니다.

설비 목록이 안 보여요

관리모드는 카테고리를 눌러야 목록이 보입니다. 또는 검색창에 설비명을 입력하세요.

앱처럼 설치하고 싶어요

브라우저 메뉴의 ‘홈 화면에 추가/설치’로 앱처럼 설치할 수 있습니다(PWA).
